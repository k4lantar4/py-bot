## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- سرور اوبونتو 22.04
- دامنه یا IP عمومی
- Docker و Docker Compose

### نصب سریع
```bash
# دریافت کد
git clone https://github.com/yourusername/mrjbot.git
cd mrjbot

# تنظیم فایل .env
cp .env.example .env
nano .env

# راه‌اندازی با Docker
docker-compose up -d

# نصب CLI
chmod +x mrjbot
sudo cp mrjbot /usr/local/bin/
```

### دستورات CLI
```bash
# نصب
mrjbot install

# آپدیت
mrjbot update

# پشتیبان‌گیری
mrjbot backup

# بررسی امتیازات
mrjbot check-points

# تنظیم لایسنس
mrjbot set-license <code>

# مدیریت سرور
mrjbot server <command>

# آمار و گزارش‌ها
mrjbot analytics <command>
```

## 🎮 تست در ویندوز

برای تست پروژه در ویندوز، مراحل زیر رو دنبال کنید:

1. نصب پیش‌نیازها:
   ```powershell
   # نصب Docker Desktop
   choco install docker-desktop
   
   # نصب Git
   choco install git
   ```

2. تنظیمات اولیه:
   - توکن ربات تلگرام رو از @BotFather دریافت کنید
   - کلید زرین‌پال رو از پنل زرین‌پال دریافت کنید
   - اطلاعات پنل 3x-UI رو آماده کنید
   - همه این اطلاعات رو در فایل `.env` قرار بدید

3. اجرای پروژه:
   ```powershell
   # اجرا با Docker
   docker-compose up -d
   
   # یا اجرای محلی
   python manage.py runserver
   ```

4. تست قابلیت‌ها:
   - ربات تلگرام: @your_bot_username
   - داشبورد: http://localhost:8000
   - API: http://localhost:8000/api/

5. نکات مهم:
   - در محیط تست، از دیتابیس تست استفاده می‌شه
   - همه پرداخت‌ها در حالت سندباکس انجام میشن
   - لاگ‌ها در `logs/` ذخیره میشن
