## 🚀 Installation & Setup

### Prerequisites
- Ubuntu 22.04 server
- Domain or public IP
- Docker and Docker Compose

### Quick Installation
```bash
# Get the code
git clone https://github.com/yourusername/mrjbot.git
cd mrjbot

# Configure .env file
cp .env.example .env
nano .env

# Start with Docker
docker-compose up -d

# Install CLI
chmod +x mrjbot
sudo cp mrjbot /usr/local/bin/
```

### CLI Commands
```bash
# Installation
mrjbot install

# Update
mrjbot update

# Backup
mrjbot backup

# Check points
mrjbot check-points

# Set license
mrjbot set-license <code>

# Server management
mrjbot server <command>

# Analytics
mrjbot analytics <command>
```
